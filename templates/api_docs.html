Currency Exchange API Documentation

Comprehensive API for real-time and historical currency exchange rates, developed by Molla Samser for Rskworld.in.

Base URL: https://yourdomain.com/api

Latest Exchange Rates

Get the latest exchange rates for a specific base currency.

GET /latest GET
Parameters
Parameter Type Description Default
base string Base currency code (3 letters) USD
Example Request
GET /api/latest?base=EUR
Example Response
{
    "base": "EUR",
    "conversion_rates": {
        "USD": 1.12,
        "GBP": 0.85,
        "JPY": 130.53,
        ...
    },
    "time_last_update_utc": "2023-06-15 00:00:00 UTC"
}

Historical Exchange Rates

Get historical exchange rates for a specific date.

GET /historical GET
Parameters
Parameter Type Description Required
date string Date in YYYY-MM-DD format Yes
base string Base currency code (3 letters) No (default: USD)
Example Request
GET /api/historical?date=2023-01-01&base=GBP
Example Response
{
    "base": "GBP",
    "date": "2023-01-01",
    "conversion_rates": {
        "USD": 1.35,
        "EUR": 1.18,
        "JPY": 152.42,
        ...
    }
}

Currency Conversion

Convert an amount from one currency to another.

GET /convert GET
Parameters
Parameter Type Description Required
from string Source currency code (3 letters) Yes
to string Target currency code (3 letters) Yes
amount number Amount to convert No (default: 1)
Example Request
GET /api/convert?from=USD&to=EUR&amount=100
Example Response
{
    "from_currency": "USD",
    "to_currency": "EUR",
    "amount": 100,
    "converted_amount": 89.50,
    "rate": 0.895,
    "timestamp": "2023-06-15T14:30:45Z"
}

List Available Currencies

Get a list of all supported currency codes.

GET /currencies GET
Parameters

No parameters required.

Example Request
GET /api/currencies
Example Response
{
    "currencies": [
        "USD",
        "EUR",
        "GBP",
        "JPY",
        ...
    ],
    "count": 170,
    "timestamp": "2023-06-15T14:30:45Z"
}
